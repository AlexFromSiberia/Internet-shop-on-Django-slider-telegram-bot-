## Интернет магазин с каруселью

В проекте реализовано:
- Telegram bot : отправка ботом уведомлений в чат о поступлении новых заявок с
    указанием Имени Клиента и его телефона. Чат id и token для бота хранятся в
    БД, что позволяет менять настройки из панели администратора;
- Разделение прав ( администратор / менеджер );
- Dotenv использован для хранения секретной информации;
- Адаптирован под мобильные устройства;
- Возможность создания / хранения в БД заявок на товары и услуги;
- Возможность присваивать заявкам статусы;
- Возможность сохранять к заявкам комментарии ( один / несколько );
- Написан с помощью Bootstrap;
- кастомизирована панель администратора;
