Классический интернет магазин с каруселью.
В проекте реализовано:

- Написан с помощью Bootstrap
- Адаптирован под мобильные устройства
- Возможность создания / хранения в БД заявок на товары и услуги
- Возможность присваивать заявкам статусы
- Возможность сохранять к заявкам комментарии(один / несколько)
- (Telegram bot) Отправка ботом уведомлений в чат о поступлении новых заявок с указанием Имени Клиента и его телефона
